# Knights of Ludd
An expansion mod for the Sector's most pious defenders.

Featuring:
- A full faction of ships, weapons and characters for the Knights
- Valiant expeditions
- Exploration storylines
- Treacherous new challenges
- Bountiful rewards
- Corporate hubris
- Modder hubris
